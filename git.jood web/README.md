# Jood - Online Perfume Store

## 📌 About the Project
Jood is an online perfume store built to provide a smooth and easy shopping experience for users who love buying perfumes online.  
The platform allows customers to browse products, place orders, and interact with the store easily.

## 🛠️ Technologies Used
- Laravel
- MySQL
- Bootstrap
- JavaScript

## ✨ Features
- User registration and login system
- Simple and easy product browsing
- Online shopping experience for customers
- Order management system
- Customer feedback and review system
- Admin dashboard for managing products and orders

## 👩‍💻 Project Purpose
The goal of this project is to simplify the online perfume shopping experience and provide a user-friendly interface for both customers and administrators.

## ⚙️ How to Run the Project
composer install  
cp .env.example .env  
php artisan key:generate  
php artisan migrate  
php artisan serve
