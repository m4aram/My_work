# UNinotify - University Notification System

## About the Project

UNinotify is a smart university notification system that improves communication between universities, students, and staff by replacing traditional messaging methods with a centralized platform.

It delivers real-time notifications and uses a priority-based system to ensure that important announcements are sent clearly and efficiently.

The application helps students stay updated instantly while enabling staff to manage official communications in an organized and reliable way.

## 📱 Technologies Used
- Flutter
- Firebase (Authentication, Firestore, Messaging)
- REST APIs (if applicable)

## ✨ Features
- User authentication (students and staff)
- Real-time notifications
- Priority-based notification system
- Organized university announcements
- Push notifications (FCM)
- Admin/staff message management
