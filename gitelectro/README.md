# Electro - E-commerce Electronics Store

## 📌 About the Project
Electro is a full-featured e-commerce web application designed for selling electronic devices such as laptops, tablets, smartphones, and accessories.  
The platform provides a smooth and user-friendly shopping experience for customers with a complete admin management system.

## 🛠️ Technologies Used
- Laravel
- MySQL
- Bootstrap
- JavaScript

## ✨ Features
- User registration and login system
- Product listing with categories (phones, laptops, tablets, accessories)
- Easy and smooth browsing experience
- Shopping cart functionality
- Order placement and management system
- Admin dashboard for managing products, categories, and orders
- Customer feedback and reviews system

## 🎯 Project Purpose
The goal of Electro is to provide a complete online shopping experience for electronic devices with an easy-to-use interface for both customers and administrators.

## ⚙️ How to Run the Project
composer install  
cp .env.example .env  
php artisan key:generate  
php artisan migrate  
php artisan serve
